# SLAM研习社

## 宗旨

1、SLAM从基础到深挖到应用，懂原理会应用，完全搞透彻，不留知识死角。

2、每个人贡献一份力量，共同构建一个互相促进的高效的学习环境，最终所有人受益。

3、如果你是小白，将获得快速入门提升的机会，如果你是优秀的学习者，可以获得持续全面的发展和提升，结识行业志同道合的朋友。
